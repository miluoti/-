Для запуска программы Вам необходимо загрузить папку "Лаб3", открыть консоль и с помощью команды cd переместиться в этот каталог. После этого введите команду python3 lab3.py. В консоли результат работы программы не отобразится, но в папке, где находилась программа, создастся файл schet2.docx по шаблону schet.docx, а также документ schet2.pdf, в которых будет находиться счет на оплату услуг для абонента.
