Для запуска программы Вам необходимо загрузить папку "Лаб1", открыть консоль и с помощью команды cd переместиться в этот каталог. После этого введите команду python3 lab2.py. В консоли результат работы программы не отобразится, но в папке, где находилась программа, создастся файл output.txt, в котором будет находиться результат тарификации абонента, и откроется дополнительное окно с графиком зависимости объема трафика абонента от времени.
