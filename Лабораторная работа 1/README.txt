Для запуска программы Вам необходимо загрузить папку "Лабораторная работа 1", открыть консоль и с помощью команды cd переместиться в этот каталог. После этого введите команду python lab1.py. В консоли результат работы программы не отобразится, но в папке, где находилась программа, создастся файл lab1.txt, в котором будет находиться результат работы программы.
